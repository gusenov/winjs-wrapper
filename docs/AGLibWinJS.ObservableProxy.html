Class: ObservableProxy

AGLibWinJS.ObservableProxy

new ObservableProxy(data, constructor)

This function is used to add binding management functionality, in which developers can bind a user-defined object to a control that is capable of notifying listeners when the value of a property changes. This is helpful for wiring up two-way data binding, something that WinJS doesn’t do itself, but isn’t too hard to pull together.
